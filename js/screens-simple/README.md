# Pantallas modo "simple" (adulto mayor)

Cada archivo `.js` exporta una función `render(container, ctx)` que pinta una
pantalla. Botones grandes, una sola acción primaria por pantalla, texto en
voseo argentino.

Pendiente:
- `inicio.js` — bienvenida + accesos a Emergencias / Familia / Médico / Cómo hago
- `emergencias.js` — 911 / SAME 107 / Bomberos 100 + botón pánico → WhatsApp con GPS
- `familia.js` — lista con foto/nombre/parentesco, llamar y WhatsApp
- `medico.js` — obra social, mail al médico (con dictado por voz), llamar
- `como-hago.js` — tutoriales paso a paso + text-to-speech + "pedir ayuda a..."
