# Pantallas modo "dashboard" (familiar)

Cada archivo `.js` exporta `render(container, ctx)`. UI más densa, controles
de gestión, formularios.

Pendiente:
- `inicio.js` — resumen del círculo, notificaciones del adulto mayor
- `miembros.js` — alta/baja, cambio de parentesco/permisos, generar invitación
- `medico.js` — editar obra_social / médico / etc.
- `contactos.js` — administrar familia + emergencias del círculo
- `notificaciones.js` — recibir pánicos vía ntfy.sh
